tauclock

A clock for camping on planet earth

tauclock_0_14.apk

NB: if you have a version before v0.4, you may need to uninstall before installing this version
=== CHANGELOG ===
  • v0.14 (tauclock_0_14.apk)
    • fixes embarrassing cache update bug
    • adds "first point of Aries" rise and set
    • daylight/darkness duration properly dodges rise/set in polar areas
  • v0.13 (tauclock_0_13.apk)
    • last of the known planet drawing bugs fixed
    • introduced body position caching to speed rendering
    • changed hilight color of inner moon
  • v0.12 (tauclock_0_12.apk)
    • omnibus bugfix for numerous drawing problems
    • known issues:
      • drawing is slower than previous versions
  • v0.11 (tauclock_0_11.apk)
    • when in "manual" offset mode, tapping left or right edge changes offset
    • algorithm changes to improve accuracy and reduce CPU load
    • minor graphic changw, draw now hand UNDER moon
  • v0.10 (tauclock_0_10.apk)
    • numerous drawing tweaks
    • initialize timezone engine in background
    • fixed some bugs updating preference settings
  • v0.9 (tauclock_0_9.apk)
    • fixed crashing bug on invalid manual locations
    • fixed crashing bug on timezone engine fail
    • fixed bug with system timezone provider when switching timezones
  • v0.8 (tauclock_0_8.apk)
    • added "Settings" page, tap in lower right to access
    • tap lower left to change timezone provider (set in settings)
    • settings can also specify offset for Julian Date
  • v0.7 (tauclock_0_7.apk)
    • fixed gimble lock problems
    • in "manual" mode, globe can be rotated with two fingers
    • changed planet colors for visibility
    • known issues:
      • no way (yet) to change time zone from system time zone
  • v0.6 (tauclock_0_6.apk)
    • fixed problems on android 8 & 9
    • made cross in manual mode more visible
    • added symbols for circumpolar planets
    • known issues:
      • no way (yet) to change time zone from system time zone
  • v0.5 (tauclock_0_5.apk)
    • select location provider by tapping in upper left
    • for manual location:
      • select "manual" in upper left
      • press and hold anywhere on dial
      • drag earth to place center cross on desired position
      • release
    • known issues:
      • for circumpolar planets above horizon, symbols are not drawn
      • no way (yet) to change time zone from system time zone
  • v0.4 (tauclock4.apk)
    • better handling on devices w/o gps
    • added dithering to twilight edges
    • changed signing key, previous key lost
